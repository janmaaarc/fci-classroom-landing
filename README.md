# FCI Classroom Landing Page

A clean, minimalist landing page for FCI Classroom - an online learning platform dedicated to mastering modern furniture design.

## Overview

This landing page serves as the main marketing site for FCI Classroom, a Skool-based community and course platform teaching furniture design principles, material selection, and interior curation.

## Features

- Responsive design (desktop, tablet, mobile)
- Smooth scroll navigation
- Interactive FAQ accordion
- Sticky header with scroll effect
- Clean, modern aesthetic with gold accents

## Tech Stack

- HTML5
- CSS3 (vanilla, no frameworks)
- JavaScript (vanilla)
- Google Fonts (Inter)

## Sections

1. **Hero** - Main headline with call-to-action buttons
2. **Stats** - Key metrics display
3. **About** - Platform introduction
4. **Modules** - Course curriculum overview
5. **Features** - Benefits of joining
6. **Pricing** - Membership details
7. **FAQ** - Common questions
8. **Instructor** - About the founder
9. **CTA** - Final call-to-action
10. **Footer** - Quick links and branding

## Getting Started

1. Clone the repository
2. Open `index.html` in your browser

No build process required - it's pure HTML, CSS, and JavaScript.

## Deployment

This static site can be deployed to:
- Netlify (drag & drop)
- Vercel
- GitHub Pages
- Any static hosting provider
